# Ennéagramme System — Les 9 Profils de Personnalité

*Basé sur "Le Grand Livre de l'Ennéagramme" de Fabien et Patricia Chabreuil (Eyrolles, 2022)*  
*Application à la création de personnages enfants 5-12 ans*

---

## 🎯 VUE D'ENSEMBLE

L'ennéagramme cartographie les **motivations** qui sous-tendent nos comportements, pas les comportements eux-mêmes. Cela en fait un outil précieux pour créer des personnages cohérents.

### Les 3 Centres d'Intelligence

```
         TÊTE (peur)
         5  6  7

CŒUR              VENTRE
(honte)           (colère)
2  3  4           8  9  1
```

| Centre | Émotion | Fonction | Question |
|--------|---------|----------|----------|
| **Ventre (Instinctif)** | Colère | Action, survie, pouvoir | "Quoi ?" / "Où ?" |
| **Cœur (Émotionnel)** | Honte | Relations, image, identité | "Qui ?" |
| **Tête (Mental)** | Peur | Analyse, sécurité, futur | "Pourquoi ?" / "Comment ?" |

### Structure de chaque profil

```
Centre préféré
    ↓
Orientation (quête fondamentale)
    ↓
Compulsion d'évitement (ego)
    ↓
Mécanisme de défense + Passion + Fixation (ego)
    ↓
Vertu + Idée supérieure + Intuition (essence)
```

## 🔷 PROFIL #1 — Le Perfectionniste

> *"Je suis droit. Je suis travailleur."*

### Caractéristiques fondamentales
- **Centre préféré** : Ventre (instinctif) dirigé vers l'**intérieur**
- **Orientation** : Définir et réaliser des idéaux élevés, rigueur personnelle
- **Fierté égotique** : Être droit, travailleur, irréprochable

### Mécanismes de l'ego
| Élément | Description |
|---------|-------------|
| **Compulsion** | Éviter à tout prix la colère (surtout la sienne) |
| **Mécanisme de défense** | Formation réactionnelle : colère réprimée, remplacée par patience forcée |
| **Passion** | Colère — présente en permanence mais refoulée |
| **Fixation** | Perfectionnisme — critique intérieur intarissable |

### Essence (état sain)
- **Vertu** : Patience — acceptation de la réalité telle qu'elle est
- **Idée supérieure** : Perfection — dans l'acceptation du réel, pas sa correction
- **Qualités** : Sage, objectif, discernant, réformateur exemplaire

### Dynamiques
- **Désintégration (stress)** → Profil #4 : mélancolie, dévalorisation
- **Intégration (croissance)** → Profil #7 : légèreté, spontanéité, joie

### Ailes
- **#1 avec aile 9 (1w9)** : Plus calme, rêveur, philosophique
- **#1 avec aile 2 (1w2)** : Plus chaleureux, soucieux d'aider

### Comportements typiques (enfant)
- Relit ses dessins plusieurs fois avant de montrer
- Corrige les autres (même amicalement)
- A du mal à déléguer
- Dit "Ce n'est pas grave" quand c'est grave
- Tensions physiques (mâchoires serrées)

---

## 💝 PROFIL #2 — L'Altruiste

> *"J'aime. J'aide."*

### Caractéristiques fondamentales
- **Centre préféré** : Cœur (émotionnel) dirigé vers l'**extérieur**
- **Orientation** : Aimer les autres, aider concrètement
- **Fierté égotique** : Être aimant, généreux, indispensable

### Mécanismes de l'ego
| Élément | Description |
|---------|-------------|
| **Compulsion** | Refuser de reconnaître ses propres besoins |
| **Mécanisme de défense** | Répression — nier ses désirs pour l'autre |
| **Passion** | Orgueil — "je n'ai besoin de rien" |
| **Fixation** | Flatterie — compliments comme outil de connexion |

### Essence (état sain)
- **Vertu** : Humilité — aimer sans attendre retour
- **Idée supérieure** : Liberté — libre de donner, libre de recevoir
- **Qualités** : Empathique vrai, altruiste authentique

### Dynamiques
- **Désintégration (stress)** → Profil #8 : agressivité, possessivité
- **Intégration (croissance)** → Profil #4 : connexion à soi, authenticité

### Comportements typiques (enfant)
- Partage ses jouets même quand il ne veut pas
- Devine ce que l'autre ressent/avant besoin
- Difficulté à dire "non"
- Frustration quand son aide n'est pas reconnue
- Peur de décevoir

---

## 🏆 PROFIL #3 — Le Battant

> *"Je réussis. Je suis efficace."*

### Caractéristiques fondamentales
- **Centre préféré** : Cœur (émotionnel) cherchant **l'équilibre**
- **Orientation** : Réussir, être valorisé par les autres
- **Fierté égotique** : Efficacité, accomplissements, image positive

### Mécanismes de l'ego
| Élément | Description |
|---------|-------------|
| **Compulsion** | Se vendre, adapter son image pour plaire |
| **Mécanisme de défense** | Identification — fusionne avec son rôle |
| **Passion** | Mensonge/Tromperie — sur soi-même d'abord |
| **Fixation** | Vanité — besoin constant de miroirs |

### Essence (état sain)
- **Vertu** : Authenticité — être vrai, pas performant
- **Idée supérieure** : Espoir — valeur intrinsèque
- **Qualités** : Réaliste, travailleur authentique, inspirant

### Dynamiques
- **Désintégration (stress)** → Profil #9 : apathie, retrait
- **Intégration (croissance)** → Profil #6 : engagement authentique, vrai soutien

### Comportements typiques (enfant)
- Montre ses bonnes notes
- Change de comportement selon l'audience
- Compétition pour être "le meilleur"
- Difficulté à reconnaître l'échec
- "Regarde ce que j'ai fait !"

---

## 🎨 PROFIL #4 — Le Romantique

> *"Je suis différent. Je suis sensible."*

### Caractéristiques fondamentales
- **Centre préféré** : Cœur (émotionnel) dirigé vers l'**intérieur**
- **Orientation** : Être soi-même, authenticité émotionnelle
- **Fierté égotique** : Sensibilité, originalité, profondeur

### Mécanismes de l'ego
| Élément | Description |
|---------|-------------|
| **Compulsion** | Créer une identité unique (différent des autres) |
| **Mécanisme de défense** | Introjection — absorbe les émotions des autres |
| **Passion** | Envie — manque de ce que les autres ont |
| **Fixation** | Mélancolie — nostalgie de ce qui n'existe pas |

### Essence (état sain)
- **Vertu** : Équanimité — calme intérieur, stabilité
- **Idée supérieure** : Origine — connexion à l'essence
- **Qualités** : Créatif inspiré, profond authentique

### Dynamiques
- **Désintégration (stress)** → Profil #2 : besoin désespéré d'amour
- **Intégration (croissance)** → Profil #1 : discipline créative, action

### Comportements typiques (enfant)
- "Personne ne me comprend"
- Créativité artistique intense
- Hauts et bas émotionnels marqués
- Attachement à ce qui est "spécial"
- Sensation d'être à côté, différent

---

## 🔬 PROFIL #5 — L'Observateur

> *"Je sais. Je comprends."*

### Caractéristiques fondamentales
- **Centre préféré** : Tête (mental) dirigé vers l'**extérieur**
- **Orientation** : Comprendre le monde, accumuler savoir
- **Fierté égotique** : Capacités intellectuelles, expertise

### Mécanismes de l'ego
| Élément | Description |
|---------|-------------|
| **Compulsion** | Retrait pour préserver énergie/reconnaitre |
| **Mécanisme de défense** | Isolement — pensée sans émotions |
| **Passion** | Avarice — peur de manquer (ressources, énergie) |
| **Fixation** | Rétention — garder l'information pour soi |

### Essence (état sain)
- **Vertu** : Détachement — non-possession du savoir
- **Idée supérieure** : Omniscience — compréhension universelle
- **Qualités** : Sage pédagogue, visionnaire clair

### Dynamiques
- **Désintégration (stress)** → Profil #7 : dispersion, addiction
- **Intégration (croissance)** → Profil #8 : action engagée, force tranquille

### Comportements typiques (enfant)
- Lecteur vorace
- Questions incessantes "pourquoi ?"
- Besoin de temps seul
- Difficulté à exprimer émotions
- Collectionneur (cartes, cailloux, savoirs)

---

## 🛡️ PROFIL #6 — Le Loyal

> *"Je suis loyal. Je fais mon devoir."*

### Caractéristiques fondamentales
- **Centre préféré** : Tête (mental) cherchant **l'équilibre**
- **Orientation** : Sécurité à travers l'allégeance (idées, groupes, personnes)
- **Fierté égotique** : Loyauté, fiabilité, engagement

### Mécanismes de l'ego
| Élément | Description |
|---------|-------------|
| **Compulsion** | Chercher autorité/cause pour se rassurer |
| **Mécanisme de défense** | Projection — attribue sa peur à l'extérieur |
| **Passion** | Peur/Lâcheté — anxiété chronique |
| **Fixation** | Doute — suspicion, besoin de confirmation |

### Essence (état sain)
- **Vertu** : Courage — affronter l'incertitude
- **Idée supérieure** : Foi — confiance dans la vie
- **Qualités** : Engagé authentique, courageux vrai

### Dynamiques
- **Désintégration (stress)** → Profil #3 : compétition nerveuse
- **Intégration (croissance)** → Profil #9 : paix intérieure, détente

### Comportements typiques (enfant)
- "Tu es sûr ?" — besoin de réassurance
- Attachement à rituels, routines
- Méfiance envers l'inconnu
- Fidélité absolue aux amis
- Questionne autorité + obéit quand rassuré

---

## 🌈 PROFIL #7 — L'Épicurien

> *"Je suis optimiste. Je suis heureux."*

### Caractéristiques fondamentales
- **Centre préféré** : Tête (mental) dirigé vers l'**intérieur**
- **Orientation** : Rendre sa vie aussi plaisante que possible
- **Fierté égotique** : Gaieté, optimisme, possibilités

### Mécanismes de l'ego
| Élément | Description |
|---------|-------------|
| **Compulsion** | Fuir vers le futur pour éviter souffrance présente |
| **Mécanisme de défense** | Rationalisation — minimise les problèmes |
| **Passion** | Gourmandise — consommation d'expériences |
| **Fixation** | Planification — toujours vers le prochain plaisir |

### Essence (état sain)
- **Vertu** : Sobriété — plaisir dans le présent
- **Idée supérieure** : Plan divin — confiance dans l'ordre
- **Qualités** : Joyeux stable, gratitude authentique

### Dynamiques
- **Désintégration (stress)** → Profil #1 : rigidité, critique
- **Intégration (croissance)** → Profil #5 : concentration, profondeur

### Comportements typiques (enfant)
- "Et si on faisait... ?" — nouvelles idées constantes
- Difficulté à finir ce qu'il commence
- Passion intense puis déclin rapide
- Fuit les moments tristes/difficiles
- "C'est pas grave !" optimisme forcé

---

## 🦁 PROFIL #8 — Le Chef

> *"Je suis fort. Je suis juste."*

### Caractéristiques fondamentales
- **Centre préféré** : Ventre (instinctif) dirigé vers l'**extérieur**
- **Orientation** : Avoir un impact, maîtriser sa vie
- **Fierté égotique** : Force, courage, justice

### Mécanismes de l'ego
| Élément | Description |
|---------|-------------|
| **Compulsion** | Éviter la faiblesse, vulnérabilité |
| **Mécanisme de défense** | Déni/Dénégation — "Non, tout va bien" |
| **Passion** | Excès — tout ou rien |
| **Fixation** | Vengeance — œil pour œil |

### Essence (état sain)
- **Vertu** : Innocence — force sans domination
- **Idée supérieure** : Vérité/Altérité — justice vraie
- **Qualités** : Leader inspirant, protecteur engagé

### Dynamiques
- **Désintégration (stress)** → Profil #5 : retrait, paranoïa
- **Intégration (croissance)** → Profil #2 : tendresse, vulnérabilité

### Comportements typiques (enfant)
- Leader naturel du groupe
- Protecteur des plus faibles
- Colère explosive mais rapide
- Difficulté à demander pardon/aide
- "Je gère" — refuse d'être aidé

---

## ☮️ PROFIL #9 — Le Médiateur

> *"Je suis bien, calme, facile à vivre."*

### Caractéristiques fondamentales
- **Centre préféré** : Ventre (instinctif) cherchant **l'équilibre**
- **Orientation** : Créer/maintien harmonie, éviter conflits
- **Fierté égotique** : Paix, stabilité, agréabilité

### Mécanismes de l'ego
| Élément | Description |
|---------|-------------|
| **Compulsion** | S'endormir/effacer pour éviter conflit |
| **Mécanisme de défense** | Narcotisation — distraction, oubli |
| **Passion** | Paresse — résistance au changement |
| **Fixation** | Indolence — pas de préférence affichée |

### Essence (état sain)
- **Vertu** : Diligence/Action — mouvement juste
- **Idée supérieure** : Amour — connexion profonde
- **Qualités** : Médiateur vrai, présence apaisante

### Dynamiques
- **Désintégration (stress)** → Profil #6 : anxiété, panique
- **Intégration (croissance)** → Profil #3 : action, engagement

### Comportements typiques (enfant)
- "Ça m'est égal" — pas de préférence
- Médiateur des disputes entre amis
- Difficulté à choisir/s'affirmer
- Peur de déranger
- Passivité masquant de la colère refoulée

## 👥 APPLICATION : CRÉER DES PERSONNAGES ENFANTS

### Règles d'or

1. **Les personnages parlent DEPUIS leur type, pas SUR leur type**
   - ❌ "Je suis comme ça parce que je suis un 7"
   - ✅ "Et si on faisait autre chose !" (fuite typique 7)

2. **Peur fondamentale active dans chaque scène** (même subtilement)

3. **Mécanisme de défense visible** :
   - #1 : colère transformée en patience forcée
   - #7 : "C'est pas grave" quand ça l'est
   - #8 : "Je gère" quand ce n'est pas vrai

4. **Niveau de santé variable** :
   - 3 = sain, connecté à vertu
   - 6 = sous pression, mécanismes égotiques
   - 9 = désintégration

5. **Micro-pas de croissance** : Pas de transformation miraculeuse. Un petit pas vers la flèche d'intégration.

### Exemple de scène : Perdre un jeu

| Profil | Ressenti | Action | Dialogue |
|--------|----------|--------|----------|
| **#1** | Colère contre soi d'avoir mal joué | Rejoue mentalement la partie | "C'était une erreur stupide." |
| **#2** | Tristesse de décevoir | Console l'autre qui a gagné | "Bravo ! Tu mérites." (sourire forcé) |
| **#3** | Honte de l'image | Minimise, change sujet | "C'était juste un jeu." |
| **#4** | Mélancolie d'être malchanceux | S'isole | "Bien sûr, ça arrive à moi." |
| **#5** | Analyse de la défaite | Se retire pour réfléchir | Silence, observation |
| **#6** | Peur du jugement | Cherche réassurance | "Tu es sûr que c'était fair ?" |
| **#7** | Fuite vers le futur | Propose autre activité | "Bon, on fait autre chose !" |
| **#8** | Colère exprimée | Conteste les règles | "C'est pas juste !" |
| **#9** | Numbing | "Ça m'est égal" | "Pas grave, on s'en fiche." |
