# Animation Screenplay — Scénario pour Animation

*Écriture de scènes, dialogue, et 12 principes Disney pour animation enfants*

---

## 1. CONSTRUCTION DE SCÈNES

### Structure d'une scène d'animation

| Élément | Fonction | Timing (30 min épisode) |
|---------|----------|-------------------------|
| **Setup visuel** | Établir lieu, personnages, ton | 5-10 sec |
| **Conflit/Objectif** | Que veut le personnage ? | 10-20 sec |
| **Obstacles** | Qu'est-ce qui bloque ? | 1-3 min |
| **Climax** | Moment de tension max | 10-30 sec |
| **Résolution** | Changement, nouvelle équilibre | 10-20 sec |

### Types de scènes

| Type | Caractéristiques | Usage |
|------|-----------------|-------|
| **Action** | Mouvement, visuel, peu de dialogue | Chases, combats, transformations |
| **Émotion** | Close-ups, silence, musique dominante | Moments de connexion, révélations |
| **Dialogue** | Verbal humor, exposition, conflit | Scènes relationnelles |
| **Transition** | Visuel fluide, changement lieu/temps | Entre actes, ellipses |

## 2. DIALOGUE EN ANIMATION

### Règles spécifiques

1. **Court et percutant** : Moins de texte que live-action
2. **Visualisable** : Ce qui est dit doit pouvoir être illustré
3. **Rythme musical** : Pacing adapté à la bande-son
4. **Répétition thématique** : Motifs qui reviennent (musicalité)

### Exemple transformation

❌ **Trop textuel** :
```
PROFIL #7
Je suis triste parce que mon ami est parti 
et je ne sais pas quoi faire sans lui.
```

✅ **Visuel** :
```
PROFIL #7 regarde le jouet que son ami lui a laissé.
Il le fait tourner entre ses doigts. Un sourire 
triste, puis il le range soigneusement.

PROFIL #7
(à lui-même)
Et si...

Il bondit, déjà vers une nouvelle idée.
```

## 3. LES 12 PRINCIPES DISNEY

### Application à l'écriture de scénario

| Principe | Définition | Application scénaristique |
|----------|-----------|---------------------------|
| **1. Squash and Stretch** | Conservation volume dans déformation | Scènes de transformation émotionnelle |
| **2. Anticipation** | Préparer l'action | Foreshadowing visuel/émotionnel |
| **3. Staging** | Clarité du plan | Une idée par scène, focalisation |
| **4. Straight Ahead / Pose to Pose** | Continuité vs poses clés | Scènes fluides vs moments forts |
| **5. Follow Through** | Inertie des mouvements | Conséquences des actions |
| **6. Slow In / Slow Out** | Accélération/décélération | Montée et descente de tension |
| **7. Arcs** | Mouvements en courbes | Arcs émotionnels des personnages |
| **8. Secondary Action** | Actions secondaires | Beats émotionnels parallèles |
| **9. Timing** | Nombre d'images | Rythme scénaristique |
| **10. Exaggeration** | Pousser le réalisme | Personnalités marquées |
| **11. Solid Drawing** | Volume et poids | Cohérence des personnages |
| **12. Appeal** | Charisme | Aimer les personnages |

### Exemple : Timing (Principe #9)

**Scène de peur** :
```
- 12 frames : Réaction normale
- 6 frames : Réaction peur (accélération)
- 3 frames : Sur-saut (choc)
- 24 frames : Suspension après choc
```

**Équivalent scénario** :
```
- Setup normal
- Accélération événement inquiétant
- Choc révélation
- Suspension émotionnelle (le temps de respirer)
```

## 4. NARRATION SANS ANTAGONISTE

### Modèles alternatifs

| Type | Structure | Exemple |
|------|-----------|---------|
| **Découverte** | Curiosité → exploration → compréhension | *March of the Penguins* |
| **Relation** | Rencontre → conflit → réconciliation | *Mon Voisin Totoro* |
| **Croissance** | Manque → apprentissage → maîtrise | *Ratatouille* (partiel) |
| **Voyage** | Départ → épreuves → retour transformé | *Le Voyage de Chihiro* |
| **Mystère** | Question → indices → révélation | *Zootopie* (sans méchant final) |

### Structure Kishōtenketsu (4 actes sans conflit)

```
Ki  (起) — Introduction : personnages et contexte
Shō (承) — Développement : expansion, approfondissement
Ten (転) — Pivot : changement de perspective (PAS conflit)
Ketsu(結) — Résolution : réconciliation, nouvelle compréhension
```

**Exemple Totoro** :
- **Ki** : Satsuki et Mei déménagent
- **Shō** : Vie quotidienne, rencontre Totoro
- **Ten** : Mei disparaît (panique)
- **Ketsu** : Totoro aide à retrouver Mei

## 5. FORMATS

### Court (5-11 min)

| Élément | Durée | Caractéristiques |
|---------|-------|-----------------|
| Setup | 30 sec | 1 personnage, 1 lieu, 1 règle |
| Inciting | 30 sec | Événement déclencheur simple |
| Escalade | 3-7 min | 3 gags/épreuves croissants |
| Climax | 30 sec | Résolution visuelle forte |
| Tag | 10 sec | Retour statu quo ou changement |

### Moyen (22-26 min / épisode série)

| Acte | Pourcentage | Contenu |
|------|-------------|---------|
| **A Story** | 60% | Intrigue principale |
| **B Story** | 30% | Sous-intrigue thématique |
| **C Story** | 10% | Running gag/continuité |

**Structure 3 actes** :
- **Acte I** (25%) : Setup, incident déclencheur
- **Acte II** (50%) : Fun & Games, Midpoint, Dark moment
- **Acte III** (25%) : Climax, résolution, leçon

### Long (45-90 min / film)

| Structure | Durée | Caractéristiques |
|-----------|-------|-----------------|
| **3 actes** | Classique | Setup, confrontation, résolution |
| **Hero's Journey** | 12 étapes | Voyage transformateur |
| **Save the Cat** | 15 beats | Comédie, blockbuster |
| **Kishōtenketsu** | 4 actes | Sans antagoniste, contemplatif |

## 6. PERSONNAGES EN ANIMATION

### Silhouette reconnaissable

Chaque personnage doit être identifiable à sa silhouette seule :
- **Profil #1** : Droiture, symétrie
- **Profil #7** : Mouvement, asymétrie joyeuse
- **Profil #9** : Ronds, douceur

### Palette émotionnelle

| Profil | Couleur dominante | Forme |
|--------|------------------|-------|
| #1 | Vert | Carrés nets |
| #2 | Rose | Courbes enveloppantes |
| #3 | Or | Éclat, miroirs |
| #4 | Violet | Formes asymétriques |
| #5 | Bleu marine | Géométrie pure |
| #6 | Gris | Formes variables |
| #7 | Jaune/Orange | Éclats, zigzags |
| #8 | Rouge | Masses imposantes |
| #9 | Vert d'eau | Ronds, flux |
