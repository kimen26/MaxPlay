# Storytelling enfant 3-4 ans

## Ce qui accroche un enfant de 3-4 ans

- Un personnage expressif qui LUI parle directement ("Tu veux m'aider ?")
- Des situations du quotidien magnifiées (le bus qui part en aventure)
- La répétition rassurante (même formule d'intro, même jingle)
- L'enfant comme héros – c'est LUI qui sauve la situation
- Humour simple : un bus qui fait une bêtise, un arrêt qu'on a oublié
- Surprise et révélation : "Oh non ! Il manque quelque chose !"

## Directions d'univers possibles – à discuter

### Option A : Ville réaliste stylisée
Ville contemporaine, bus qui ressemblent à de vrais bus (RATP, TCL...).
Tom reconnaît des éléments de son quotidien. Fort ancrage affectif.
Personnage guide : un chauffeur sympa ou le bus lui-même qui parle.

### Option B : Monde fantastique des bus
Les bus ont une vie secrète la nuit. Ils parlent, ont des personnalités.
Plus imaginatif, permet plus de liberté créative.
Personnage guide : un bus vedette avec une forte personnalité.

### Option C : Documentaire interactif
Pas de fiction, mais des "vrais" faits sur les bus présentés ludiquement.
"Sais-tu pourquoi le bus 21 est rouge ?" → Tom clique pour découvrir.
Convient si Tom aime les faits réels plus que la fiction.

## Règles d'écriture pour 3-4 ans

- Phrases : 5-8 mots max
- Vocabulaire : concret, familier, pas d'abstraction
- Rythme : lent, avec pauses – laisser Tom réagir
- Questions : "Tu vois quelle couleur ?" (ouvert) ou "C'est le 7 ou le 4 ?" (guidé)
- Ton : enthousiaste mais calme, jamais stressant

## Structure d'une micro-histoire de mini-jeu

```
ACCROCHE (5 sec) : situation problème simple et claire
→ "Oh non ! Les bus ont perdu leurs garages !"

MISSION (2 sec) : ce que Tom doit faire
→ "Tu peux aider chaque bus à retrouver le sien ?"

PENDANT : encouragements ponctuels, pas de monologue
→ "Bien vu !" / "Regarde la couleur !"

RÉSOLUTION : victoire célébrée avec le personnage
→ "BRAAAVO ! Tu as tout retrouvé ! Les bus sont contents !"
```

## Format de proposition de personnage

```
## Nom du personnage
Type : bus / humain / animal / autre
Apparence : ...
Voix / ton : ...
Rôle dans le jeu : guide / antagoniste bienveillant / ami
Phrase signature : "..."
Expression de joie : ...
Expression d'encouragement : ...
```
