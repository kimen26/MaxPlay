# Fondements de l'Ennéagramme

## Origine

L'Ennéagramme est un système de 9 types de personnalité organisés en cercle, reliés par
des lignes internes. Ses origines sont multiples :
- Traditions mystiques (soufisme, Gurdjieff)
- Psychologie moderne (Oscar Ichazo années 1960, Claudio Naranjo années 1970)
- Formalisé en Occident par Don Richard Riso et Russ Hudson (années 1980-2000)

---

## Les 3 Centres (Triades)

Chaque type appartient à un centre d'intelligence dominant :

### Centre Instinctif / Corps (Types 8, 9, 1)
- Émotion centrale : **Colère** (exprimée, refoulée, ou niée)
- Question : *Ai-je le droit d'exister ? D'agir ?*
- Type 8 : exprime la colère
- Type 9 : endort la colère
- Type 1 : refoule la colère → devient ressentiment

### Centre Émotionnel / Cœur (Types 2, 3, 4)
- Émotion centrale : **Honte / Tristesse**
- Question : *Suis-je aimable ? Ai-je de la valeur ?*
- Type 2 : refoule la honte → se rend indispensable
- Type 3 : endort la honte → se construit une image
- Type 4 : plonge dans la honte → cherche son identité

### Centre Mental / Tête (Types 5, 6, 7)
- Émotion centrale : **Peur / Anxiété**
- Question : *Est-ce que je suis en sécurité ? Ai-je les ressources ?*
- Type 5 : refoule la peur → accumule le savoir
- Type 6 : exprime la peur → cherche du soutien
- Type 7 : fuit la peur → cherche de nouvelles expériences

## Les Lignes d'Intégration et Désintégration

En période de **croissance (intégration)**, chaque type adopte les traits positifs d'un autre type.
En période de **stress (désintégration)**, chaque type adopte les traits négatifs d'un autre type.

| Type | Désintégration (stress →) | Intégration (croissance →) |
|------|---------------------------|----------------------------|
| 1    | → 4 (mélancolique, replié) | → 7 (spontané, joyeux) |
| 2    | → 8 (agressif, dominateur) | → 4 (introspectif, créatif) |
| 3    | → 9 (apathique, déconnecté) | → 6 (loyal, coopératif) |
| 4    | → 2 (clinquant, possessif) | → 1 (ordonné, principié) |
| 5    | → 7 (dispersé, impulsif) | → 8 (affirmé, décisif) |
| 6    | → 3 (compétitif, arrogant) | → 9 (relâché, confiant) |
| 7    | → 1 (perfectionniste, critique) | → 5 (focalisé, profond) |
| 8    | → 5 (retiré, isolé) | → 2 (généreux, chaleureux) |
| 9    | → 6 (anxieux, paranoïaque) | → 3 (engagé, efficace) |

## Les Sous-types (Instincts)

Chaque type se décline en 3 variantes selon l'instinct dominant :

### Conservation (CO)
- Centré sur la survie, les ressources, le confort personnel
- *"Ai-je assez ? Suis-je en sécurité ?"*

### Social (SO)
- Centré sur la place dans le groupe, la reconnaissance collective
- *"Où est ma place ? Suis-je inclus/respecté ?"*

### Sexuel / Tête-à-tête (SX)
- Centré sur l'intensité, la fusion, la connexion profonde 1-à-1
- *"Suis-je vibrant ? Ai-je une vraie connexion ?"*

## Les 9 Niveaux de Santé (Riso-Hudson)

Pour chaque type, 9 niveaux du plus sain au moins sain :

**Niveaux de santé :**
- **1** : Libéré, illuminé — la vertu s'exprime pleinement
- **2** : Psychologiquement fort — la personnalité au service de la vie
- **3** : Socialement adapté — fonctionnel, épanoui dans ses relations
- **4** : Normal — adaptatif mais avec des habitudes mécaniques
- **5** : Interzone — le comportement défensif commence
- **6** : Surchargé — les défenses s'intensifient, conflit avec l'environnement
- **7** : Réactif — comportements involontaires, problèmes relationnels
- **8** : Compulsif — perte partielle de contrôle
- **9** : Pathologique — rupture avec la réalité
