## 📚 Comment utiliser ces ressources

### Pour écrire une nouvelle scène

```
ÉTAPE 1 : Choisir le personnage central
→ Consulter : docs/Eneagramme/personnages/type-XX-nom.md

ÉTAPE 2 : Choisir la situation (si universelle)
→ Consulter : docs/Eneagramme/situations/emotions-universelles.md

ÉTAPE 3 : Vérifier le niveau de santé
→ Consulter : docs/enneagramme-9-niveaux-riso-hudson.md

ÉTAPE 4 : Structurer la scène
→ Consulter : Skill narrative-craft (template, beats)

ÉTAPE 5 : Écrire avec les techniques
→ Consulter : Skill narrative-storytelling

ÉTAPE 6 : Vérifier la cohérence
→ Consulter : docs/Eneagramme/ressources/guide-auteur.md
```

### Pour créer un nouvel épisode

1. **Choisir le thème de passion de Léo** (botanique, volcans, etc.)
2. **Sélectionner les personnages présents** selon la saison
3. **Identifier le nœud émotionnel** qui teste un personnage
4. **Appliquer le template d'épisode** (5 étapes)
5. **Chercher l'opportunité de croissance** (flèche d'intégration)

### Pour analyser un personnage de référence

→ Consulter : `docs/analyse-personnages-manga-enneagramme.md`

Exemple : comment Naruto (type 7) gère la perte de Jiraya → inspiration pour comment Léo pourrait gérer une perte.

## 📝 Note finale

Ce projet est une **infrastructure narrative complète** — pas seulement des personnages, mais tout un système pour les faire vivre de façon cohérente et profonde.

Chaque scène écrite avec ces outils sera :
- **Psychologiquement juste** (basée sur l'ennéagramme)
- **Narrativement solide** (structure éprouvée)
- **Émotionnellement authentique** (situations de vie réelles)
- **Adaptée à l'enfance** (langage, thèmes, rythme)

**Le système est prêt. À toi d'écrire les histoires.** 🌟
