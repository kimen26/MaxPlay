## ⚠️ INSTRUCTIONS DE TÉLÉCHARGEMENT

### Pixabay (le plus simple)
1. Cliquer sur le lien
2. Cliquer sur "Télécharger" (pas besoin de compte)
3. Choisir MP3 ou WAV
4. Renommer avec un nom simple (ex: `coin.mp3`)

### Freesound.org (nécessite compte gratuit)
1. Créer un compte gratuit
2. Cliquer sur le lien
3. Cliquer sur "Download"
4. Convertir en MP3 si besoin (WAV = gros fichier)

### Creazilla (sans compte)
1. Cliquer sur le lien
2. Cliquer sur "Download"
3. Choisir le format

## 🔧 CONVERSION AUDIO

Si tu as des fichiers WAV trop gros :
**FFmpeg (gratuit):**
```bash
ffmpeg -i input.wav -codec:a libmp3lame -qscale:a 2 output.mp3
```

**Ou utiliser:** https://convertio.co/fr/wav-mp3/ (en ligne, gratuit)
