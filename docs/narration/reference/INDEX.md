## Règle

- **Ne pas distiller ces fichiers dans les fiches canon.** Ce sont des catalogues d'inspiration, pas de la matière d'univers.
- Si une analyse déclenche une idée → direction **INBOX.md** avec référence (ex: "cf. reference/analyse-personnages-manga...md §Naruto").
- Aucun de ces fichiers ne contraint les personnages du projet : Wex & co n'ont pas à coller à un exemple.
