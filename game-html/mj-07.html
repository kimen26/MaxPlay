🗺️ La journée de Max

Explore Villejuif avec Max !

  • Marche jusqu'à l'arrêt de bus
  • Prends le bon bus (185, V7, M7...)
  • Voyage vers l'école, chez Valouette...
  • 3 missions à compléter !

Ce jeu utilise Phaser.js et nécessite un serveur local.
Ouvre un terminal dans le dossier game/ et lance :

npm install && npm run dev
